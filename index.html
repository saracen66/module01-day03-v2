Learning Git & Github

Commands for new repository in Github

There are 6 commands to create a repository in github. The commands are:

  1. git init
  2. git add .
  3. git commit -m "meaningful comment"
  4. git branch -M main
  5. git remote add origin https://......
  6. git push -u origin main

Submitting codes and files after small changes

Every time we make some changes on our code, we need to resubmit all the files and codes to github. To do that we need to add three commands into our terminal or command promt. The codes are:

  1. git add .
  2. git commit -m "meaningful comment"
  3. git push
